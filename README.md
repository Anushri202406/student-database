# Student Database - SQL Project 🎓

Ye SQL ka chota sa project hai jisme 2 tables hain: students aur marks.

## Files kya karti hain?
1. `create_tables.sql` - Tables banata hai
2. `insert_data.sql` - Data daalta hai  
3. `queries.sql` - Data nikalta hai

## Kaise chalaye?
1. VS Code me SQLite extension install karo
2. Pehle create_tables.sql chalao
3. Phir insert_data.sql chalao
4. Phir queries.sql chalao
